Как работать быстрее
1. Каталоги по запросу
Загружайте каталоги только для выбранных площадок кнопкой "Загрузить каталоги". Это снижает стартовую нагрузку и ускоряет UI.
2. Кэш 24 часа
Категории и деревья кэшируются в API и браузере. Принудительное обновление делайте только при необходимости.
3. Два уровня анализа
Сначала overview по нескольким площадкам, затем открывайте страницу конкретной площадки для глубоких графиков и таблиц.
4. Отладка покрытия
Смотрите блок "Покрытие и ограничения": lower-bound, mixed-currency и загрузку по разделам.
